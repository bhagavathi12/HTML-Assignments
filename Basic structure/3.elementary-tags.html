FrontEnd

Frontend development involves creating the visual and interactive elements of a website or application that users interact with directly.

HTML

HTML (HyperText Markup Language) is the standard language used to create and design web pages and web applications.


CSS

CSS (Cascading Style Sheets) is used to style and layout web pages, controlling elements like colors, fonts, and spacing.


JS

JavaScript is a programming language that enables interactive and dynamic features on websites, such as form validation and animations.


REACT

React is a JavaScript library for building user interfaces by creating reusable UI components and managing the state of applications efficiently.


Redux toolkit

Redux is a state management library for JavaScript applications that helps manage and predict state changes in a consistent and predictable manner.
